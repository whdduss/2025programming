# IT 직업 정보 비교 플랫폼

IT 관련 직업들의 정보를 한 눈에 보고, 연봉, 업무환경 등을 비교할 수 있는 웹사이트입니다.

## 주요 기능

### 1. 직업 정보
- 모든 IT 관련 직업들의 목록을 확인할 수 있습니다
- 직업 이름을 클릭하면 상세 정보(연봉, 성장 가능성, 업무환경 등)를 볼 수 있습니다

### 2. 직업별 비교
- 여러 직업을 선택하고 카테고리를 선택하여 비교할 수 있습니다
- 바 차트 형식으로 시각적으로 비교 결과를 확인할 수 있습니다
- 최소 2개 이상의 직업과 1개의 카테고리를 선택해야 비교가 가능합니다

### 3. 나의 직업 찾기
- 간단한 설문에 답변하여 본인에게 가장 적합한 IT 직업을 찾을 수 있습니다
- ChatGPT API를 활용하여 설문 결과를 분석하고 직업을 추천합니다

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. API 키 설정 (선택사항):
- `.streamlit/secrets.toml.example` 파일을 참고하여 `.streamlit/secrets.toml` 파일을 생성합니다
- `OPENAI_API_KEY`에 실제 OpenAI API 키를 입력합니다
- API 키가 없어도 기본 추천 기능을 사용할 수 있습니다
- ⚠️ **중요**: `secrets.toml` 파일은 `.gitignore`에 포함되어 있어 GitHub에 업로드되지 않습니다

3. 앱 실행:
```bash
streamlit run app.py
```

## 사용 방법

1. **직업 정보**: 사이드바에서 "직업 정보"를 선택하고, 직업 목록에서 원하는 직업을 클릭합니다.

2. **직업별 비교**: 
   - "직업별 비교"를 선택합니다
   - 비교하고 싶은 직업들을 체크박스로 선택합니다 (최소 2개)
   - 비교할 카테고리를 선택합니다
   - "비교하기" 버튼을 클릭합니다

3. **나의 직업 찾기**:
   - "나의 직업 찾기"를 선택합니다
   - 모든 설문 질문에 답변합니다
   - "직업 찾기" 버튼을 클릭합니다
